analagous()method 
public static function analagous(c:uint, angle:Number = 0.02777, contrast:Number = 0.25):ColorPalette

Returns colors that are next to each other on the wheel. These yield natural color schemes (like shades of water or sky). The angle determines how far the colors are apart, making it bigger will introduce more variation. The contrast determines the darkness/lightness of the analogue colors in respect to the given colors.

Parameters
c:uint — a base color
 
angle:Number (default = 0.02777) — how far the colors are apart between 0-1, default=0.02777 (5 degrees)
 
contrast:Number (default = 0.25) — the darkness/lightness of the analogue colors with respect to the base colors, default=0.25

Returns
ColorPalette
