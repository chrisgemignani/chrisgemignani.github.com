Classpublic class FunctionSequence
InheritanceFunctionSequence Sequence Transition flash.events.EventDispatcher

Transition that runs sub-transitions in sequential order while also invoking a function before each sub-transition is run. Each function must take a Transition instance (often a Transitioner) as input. Function sequences can only be played forwards; any attempt to play one in reverse will result in an error.

Function sequences are useful for ensuring a particular function is run before a sub-transition begins. For example, the function may populate the values of a Transitioner on the fly, or may be used to control other variables or side-effects that affect the subsequent sub-transition(s).
