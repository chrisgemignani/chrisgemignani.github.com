Packageflare.vis.operator.layout
Classpublic class TreeMapLayout
InheritanceTreeMapLayout Layout Operator

Layout that places node in a TreeMap layout that optimizes for low aspect ratios of visualized tree nodes. TreeMaps are a form of space-filling layout that represents nodes as boxes on the display, with children nodes represented as boxes placed within their parent's box. This layout determines the area of nodes in the tree map by looking up the sizeField property on leaf nodes. By default, this property is "size", such that the layout will look for size values in the DataSprite.size property.

This particular algorithm is taken from Bruls, D.M., C. Huizing, and J.J. van Wijk, "Squarified Treemaps" In Data Visualization 2000, Proceedings of the Joint Eurographics and IEEE TCVG Sumposium on Visualization, 2000, pp. 33-42. Available online at: http://www.win.tue.nl/~vanwijk/stm.pdf.

For more information on TreeMaps in general, see http://www.cs.umd.edu/hcil/treemap-history/.
