Interfacepublic interface IOperator extends mx.core.IMXMLObject
ImplementorsOperator, OperatorList, SpanningTree

Interface for operators that perform processing tasks on the contents of a Visualization. These tasks include layout, and color, shape, and size encoding. Custom operators can be defined by implementing this interface;
