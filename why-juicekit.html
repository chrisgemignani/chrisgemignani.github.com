Why JuiceKit?
How does JuiceKit compare to other Flash-based visualization Libraries

Why JuiceKit? JuiceKit is not the only choice for visualization. Here are some alternatives

Name
Description
    IBM ILog Elixir A commercial package of visualization components
    Birdeye An open-source project to bring visualization to Flex
    Axiis An open-source framework built on Degraffa and Flex data binding
    Flare An open-source ActionScript framework that implements many modern visualizations.
    Flex charting The built-in charting components that are a part of Flex Professional
    flextreemap An open source treemap component by Flex evangelist Josh Tynjala

Let's review the key design features of JuiceKit and the areas where we feel it is a better choice than specific other options.

Speed

ilog birdeye axiis flex2treemap

Speed matters, and JuiceKit allows you to see more data in ways that other solutions can’t.

Managing data

ilog birdeye flare flex flex2treemap

JuiceKit makes it easy to start working with data.

Creating new visualizations

ilog flex2treemap

JuiceKit builds on the declarative open source framework Flare for visualizations.

Open source

ilog flex

Adobe Flex is open source and JuiceKit is a perfect complement.

Variety of visualizations
